Predicción de un valor numérico. Valor de una casa.¶

Algoritmo: regresión lineal¶

Un ejemplo de aplicacin del Aprendizaje Automatizado (Machine Learning) Supervisado es la prediccin de un valor numrico por medio de un algoritmo conocido como "regresin" o regresin lineal.

En el caso presente, se tiene un conjunto de datos de valor en dlares de casas de Boston. Por medio de los costos conocidos de casas basados en varias caractersticas, se debe predecir el costo de una casa que no se encuentra en el conjunto de datos inicial.

Es decir, utilizando este conjunto de datos, se desea realizar un modelo que permita predecir el valor de otras casas que no se encuentran en el conjunto de datos.

Este es un ejemplo tradicional del Aprendizaje Automatizado Supervisado, en donde se cuenta con diversas caractersticas conocidas y el valor de un producto. Por medio de estos datos se desea establecer un mecanismo de prediccin para otros productos con mismo conjunto de caractersticas, pero que no se conozca su valor. Es decir, deseamos predecir el valor de un producto que no se encuentra en los anteriores. Este es el problema de la regresin lineal multivariada o mltiple clasico de la Estadistica.

La regresin lineal mltiple permite:

identificar que variables independientes (causas) explican una variable dependiente (resultado)

predecir valores de una variable, es decir, a partir de unas caractersticas predecir de forma aproximada un comportamiento o estado

El conjunto de datos de casas de Boston para predicción del precio¶

El conjunto de datos tiene un total de 506 registros, repartidos como 404 datos de entrenamiento y 102 datos para la evaluacin del modelo.

Las caractersticas (features) de los registros miden aspectos diferentes de la casa, por ejemplo tasa de criminalidad, tasa de impuestos locales de predial, y otros. En el caso presente se tienen 13 caractersticas para cada casa y un precio en dlares del valor de la casa.

El objetivo del modelo es predecir el precio promedio de las casas en el mismo vecindario, dados algunos parmetros o caractersticas (features) como la tasa de crimenes (crime rate), la tasa impositiva del predio (local property tax rate), y otras caractersticas similares.

Cada una de estas caractersticas tiene un rango diferente de valores y medidas diferentes, por lo que segn las mejores prcticas de la estadstica, se requiere normalizar en una primera etapa, todas las caractersticas a valores entre 0 y 1.

En este contexto se tiene un nmero reducido de puntos, en total 506 con 404 de entrenamiento y 102 para evaluacin. Esto nos sugiere segn las mejores prcticas utilizar un mtodo de pruebas y evaluacin conocido como Validacion cruzada con k elementos ("K-fold cross-validation").

Los datos del proyecto se encuentran en la biblioteca keras y se obtienen por medio de una funcin load_data() que se encuentra en el modulo keras.datasets

Preparación de los datos de entrada¶

Como se observa, las caractersticas (features) de los puntos en los conjuntos de datos toman valores en diferentes escalas. Por lo que deben homogeneizarse con un proceso que denominamos Preparacin de los datos de entrada

La preparacion de los datos es una de las actividades estratgicas en la Inteligencia Artificial y en general en la Ciencia de Datos. Esto es esencial para que los modelos matemticos que se construyen, tengan resultados con significado.

Dada la diversidad de rangos y medidas diferentes de las caractersticas, un buena prctica en estadistica es normalizar las caractersticas, tomando por cada caracterstica (una columna en los datos de entrada), la media (mean) de sus valores, restarla al valor actual y dividir el resultado por la desviacin estndar.

Por medio de este procedimiento, el valor de la caracterstica esta centrada alrededor del valor 0 y tendr una desviacin estndar unitaria.

Para realizar esta normalizacin se utiliza la biblioteca Python Numpy.

Definición Red Neuronal con keras¶

El concepto base de una red neuronal es la capa (layer).

En la construccin de una red neuronal se debe definir cada una de las capas (layer).

Una capa es un modulo de procesamiento de datos (data-processing module) que puede pensarse como un filtro que recibe datos y los transforma en otra representacin que aporta valor a su entendimiento y a la solucin de la problemtica.

Algunos datos entran en la capa y salen otros de la capa, con una representacin de mayor utilidad para la solucin del problema. (Some data comes in, and comes out in a more useful form).

El concepto del Aprendizaje Profundo se refiere a encadenar capas sencillas que instrumentan una especie de proceso progresivo de "destilacin de datos" (progressive "data distillation").

En el caso presente, la red neuronal consiste de una secuencia de dos capas Dense, con funcin de activacin "relu", Rectified Linear Unit.

Una capa tipo Dense tambien se conoce como capa neuronal totalmente conectada ("fully-connected neural layer"). Es decir, todas las neuronas de esta capa se conectan con las neuronas de la siguiente capa.

Se define el tipo de red: Sequential()

Se aade cada una de las capas (layer).

La primera capa con tipo Dense(), nmero de neuronas en la capa, en el caso presente tenemos, 64

La funcin de activacin de la capa activation='relu', y el tipo de forma de la entrada de los datos que se entregan a la red: input_shape='784'

La segunda capa con tipo Dense(), nmero de neuronas en la capa (64), la funcin de activacin de la capa: activation='relu'

La tercera capa tiene 1 neuronas sin funcin de activacin.

La red neuronal al final tiene una sola neurona y no contiene funcin de activacin.

Este es el caso de una capa lineal o regresion lineal escalar en donde se debe establecer una prediccin numrica de un solo valor continuo (predict a single continuous value).

Observamos que estamos compilando la red con una funcin de perdida mse (Mean Squared Error).

La funcin de prdida mse es el cuadrado del error, es decir el cuadrado de la diferencia entre las predicciones y el objetivo (targets).

Este es una funcin de error ampliamente utilizada en problemas de regresin lineal.

En la etapa de entrenamiento se utiliza otra mtrica: mae (Mean Absolute Error).

Error Absoluto Promedio. El error absoluto promedio es una medida comn del error de pronstico en anlisis de series de tiempo.

$\mathrm{MAE}=\frac{\mathrm{SAE}}{N}=\frac{\sum_{i=1}^N \left | x_i - \hat x_i \right |}{N}$

SAE es la suma de los errores absolutos (o desviaciones)

{$x_i$} son la observaciones actuales de las series de tiempo

{$\hat x_i$} es la serie de tiempo estimada o pronosticada

Validación del modelo utilizando la validación de tipo K-fold (K-fold validation)¶

Para evaluar la red neuronal en la etapa de ajuste de sus parmetros (por ejemplo el nmero de pocas utilizadas en el entrenamiento), podemos de forma sencilla partir los datos en un conjunto de entrenamiento (training set) y un conjunto de validacin o pruebas (validation set)

Sin embargo, dado que se cuenta con un nmero limitado de puntos, el conjunto de pruebas o validacin sera muy pequeo.

En circunstancias similares, una de las mejores prcticas en Estadstica es utilizar un mtodo de validacin denominado Validacion cruzada con k elementos ("K-fold cross-validation")

El mtodo consiste en partir el conjunto de datos disponibles en K particiones, generalmente se selecciona K=4 o K=5.

Una vez construidos K modelos idnticos y habiendo realizado el entrenamiento de cada uno con K - 1 particiones y entrenando con la particin restante, el score de validacin del modelo es el promedio de los scores de las K validaciones obtenidas

En trminos de cdigo, tenemos lo siguiente:

Acceso a la historia de entrenamiento modelo en Keras¶

Keras proporciona la capacidad de registrar devoluciones de llamada cuando se entrena un modelo de aprendizaje profundo.

Una de las devoluciones de llamada predeterminadas que se registra cuando se entrena a todos los modelos de aprendizaje profundo es la devolucin de llamada del historial.

Registra mtricas de entrenamiento para cada poca.

Esto incluye la prdida y la precisin (para problemas de clasificacin), as como la prdida y la precisin para el conjunto de datos de validacin, si se establece uno.

El objeto histrico se devuelve de las llamadas a la funcin fit() utilizada para entrenar el modelo.

Las mtricas se almacenan en un diccionario en el miembro histrico del objeto devuelto.

def smooth_curve(points, factor=0.9):
  smoothed_points = []
  for point in points: 
        #Esto nos sugiere segun las mejores practicas utilizar un metodo de pruebas y 
        #evaluacion conocido como Validacion cruzada con k elementos 
        ##("K-fold cross-validation").
    if smoothed_points:
      previous = smoothed_points[-1]
      smoothed_points.append(previous * factor + point * (1 - factor))
    else:
      smoothed_points.append(point)
  return smoothed_points

En resumen, de este ejemplo tenemos:¶

  • La regresin se realiza utilizando diferentes funciones de prdida (loss functions) para llevar a cabo la clasificacin; Mean Squared Error (MSE) es una funcin de prdida comnmente utilizada para regresin.
  • Del mismo modo, las mtricas de evaluacin que se utilizan para la regresin difieren de las utilizadas para la clasificacin; naturalmente, el concepto de "precisin" no aplica para la regresin. Una mtrica de regresin comn es el Error Absoluto Medio (MAE Mean Absolute Error).
  • Cuando las caractersticas en los datos de entrada tienen valores en diferentes rangos, cada caracterstica se debe normalizar de forma independiente como un paso de preproceso de los datos.
  • Cuando hay pocos datos disponibles, el uso de la validacin K-Fold es una gran herramienta para evaluar un modelo de manera confiable (reliably evaluate a model).
  • Cuando hay pocos datos de entrenamiento disponibles, es preferible usar una red pequea con muy pocas capas ocultas (generalmente solo una o dos), para evitar un sobreajuste severo (avoid severe overfitting).
